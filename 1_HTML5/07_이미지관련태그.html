img 태그

-웹문서에 사진, 그림 같은 이미지 삽입하는 용도의 태그

[속성]
1) src:삽입할 이미지 경로를 지정하는 속성(파일경로,웹주소)
2) width / height:이미지 크기 지정하는 속성
3) alt:이미지 설명하는 속성->이미지 출력되지 않는 경우 화면에 직접 표시

src 속성확인

width, height 속성을 이용한 크기조절(고정/가변크기)

고정크기단위(px):화면크기와 상관없이 지정된 값의 크기를 갖는 단위

가변크기단위(%):화면이나 영역기준으로 사이트 크기에 따라 이미지 크기 변경되는 단위

alt 속성 확인
